About

The Near-Earth Object Statistical Management Program (NEOSMAP) was built for the University of Washington NEO observation follow-up team to aid in object tracking and selection.

If you find any bugs, or have any suggestions or criticism you would like to share, please feel free to raise an issue on the project GitHub repository.
